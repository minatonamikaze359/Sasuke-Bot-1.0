A simple WhatsApp bot using **Baileys**.

---

### 🔥 Features
- Auto QR login
- `.menu` command showing **all commands**
- AI Assistant (`.gpt`)
- Custom owner + author
- Sends bot logo

---

### 📜 Available Commands
Use `.menu` or `.help` in WhatsApp to view **all command categories**:
- 🌐 General
- 👮 Admin
- 🔒 Owner
- 🎨 Image/Sticker
- 🎮 Games
- 🤖 AI
- 🎯 Fun
- 🔤 Textmaker
- 📥 Downloader
- 💻 Github
